# 创建测试高德地图基本功能的脚本

## 目标

创建一个 `test_amap.py` 脚本，测试使用高德地图调用展示地图、添加地图控件、添加点标记等基本功能。

## 实现步骤

### 1. 创建新文件 `test_amap.py`

* 在 `d:\GPS-Seeder\gps_rover` 目录下创建新文件

* 导入必要的模块：`sys`、`os`、`PySide6.QtWidgets`、`PySide6.QtWebEngineWidgets`、`PySide6.QtCore`

### 2. 实现地图显示功能

* 使用 Qt WebEngineView 加载 HTML 内容

* 使用高德地图 JS API 创建地图实例

* 设置地图中心点和缩放级别

* 使用项目提供的高德地图 key 和安全密钥

### 3. 实现地图控件功能

* 添加缩放控件

* 添加比例尺控件

* 添加地图类型切换控件

* 添加定位控件

### 4. 实现点标记功能

* 在地图上添加标记点

* 为标记点添加弹窗信息

* 实现标记点的点击事件

### 5. 实现其他基本功能

* 实现地图点击事件，显示点击位置的坐标

* 实现地图移动事件，更新中心点坐标显示

* 添加简单的控制面板，用于测试不同功能

### 6. 测试和调试

* 运行脚本，测试地图加载是否正常

* 测试地图控件是否正常工作

* 测试点标记功能是否正常

* 调试可能出现的问题

## 技术要点

* 使用 PySide6 的 QWebEngineView 来显示地图

* 使用 HTML 和 JavaScript 来调用高德地图 JS API

* 使用项目提供的高德地图 key 和安全密钥

* 实现 Python 和 JavaScript 之间的通信

* 确保代码符合项目的代码风格和规范

## 预期结果

* 成功创建 `test_amap.py` 脚本

* 脚本能够正常运行，显示高德地图

* 能够添加和管理地图控件

* 能够添加和管理点标记

* 能够响应地图事件

## 注意事项

* 确保使用正确的高德地图 key 和安全密钥

* 确保网络连接正常，以便加载高德地图 JS API

* 确保代码符合项目的代码风格和规范
