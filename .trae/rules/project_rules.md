# 注意事项
- 项目是一套上位机系统，需要通过传感器采集定位、加速度等实时高频数据，进行解析、计算、编排等，将计算结果展示在地图上，并且通过串口上Modbus等协议，向下位机发送指令
- 项目对于数据的处理，要确保真实、有效、高精度，不要伪造数据，不要轻易的损失数据精度，避免对定位结果造成错误，影响后续的操作

- 本项目使用Qt6框架构建，需要符合Qt6的编码规范，UI文件为src/ui/mapwidget.ui，主窗口代码为src/ui/mainwindow.cpp
- 开发的第一性原则，如无必要，勿增实体，保持代码整洁、注释清晰、命名规范
- 本项目通过CMakeLists.txt配置，需要在Qt6的开发环境中编译运行，GPSSeeder.pro文件为Qt6的项目配置文件

- 项目由用户手动编译运行调试并反馈bug，不需要在推理上下文中运行调试等指令
- 项目运行的程序需要确保性能得到充分优化，避免内存泄漏、CPU占用过高等问题，并且需要避免长时运行可能带来的其他隐含问题
