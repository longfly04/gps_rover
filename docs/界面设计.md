# 高精度农业自动导航与作业控制界面设计

## 主窗口 tab 页
上半部分：
- 显示地图，默认使用osm地图，默认使用卫星底图
- 如果GPS正常工作，则在地图上实时显示设备的经纬度位置，并自动将地图中心定位到该位置
- 在地图上，显示设备运行的轨迹
- 在地图上，显示当前正在作业的小区的边界框
- 在地图上，显示规划的路线
- 在地图下方，显示经过校准的本地时间
- 在地图下方，显示IMU惯性数据，包括加速度、角速度、磁场方向角、角度等
- 在地图下方，显示GPS定位数据，包括经纬度、高度、速度

下半部分：
- 用红绿灯显示是否正在进行播种
- 当前作业下的需要播种的小区的滚动列表，滚动5行显示，对已经播种完毕、当前正在播种、下一个播种的小区，分别用绿色、橙色、黄色标记，动态变化实时刷新
- 显示路径规划数据，包括当前区间、目标点、触发点
- 显示作业控制数据，包括播种状态、 作业进度等

## 配置窗口 tab 页
- 支持用红绿灯显示连接的IMU模块和GPS的状态
- 支持设置IMU参数，包括串口号，波特率，下拉列表需要读取当前系统的串口列表。设置串口之后，点击开启按钮，即可连接IMU模块，点击关闭按钮，就可以断开连接
- 支持设置GPS参数，包括串口号，波特率，下拉列表需要读取当前系统的串口列表。设置串口之后，点击开启按钮，即可连接GPS模块，点击关闭按钮，就可以断开连接
- 支持设置路径规划参数，包括触发点距离、安全余量、行距、过道宽度、小区长度、提前量等数值
- 允许用户自定义参数，包括地图类型、IMU参数、GPS参数、传感器参数、路径规划参数、作业控制参数
- 支持通过文件加载和保存设备配置参数，参数的格式是json格式

## 数据窗口 tab 页
- 配置sqlite数据库连接，数据库文件路径是./data/rover.db
- 通过读取数据库，可以获取历史作业数据的列表
- 点击历史作业数据列表中的项，即可显示该次作业的概要信息
